Это парсер сайта https://vk.com/@yvkurse

После запуска нужно ввести название документа excel
после работы программы в папке result появится документ excel с 
вашим названием и с полным описанием каждого поста

Могло быть недопанимание в самом задании, но мой парсер заходит 
на каждый сайт из поста и выдаёт первую строку информации с этого сайта

всё реализовано на языкке python
подключены библиотеки request, bs4

также используется вк api
